1.-Creamos una carpeta y la añadimos a Visual Studio Code.
2.-Creamos el archivo Readme.md
3.-Creamos los archivos citados en el enunciado de la actividad.
4.-Nos logueamos en nuestra cuenta de Git.
5.-Creamos un nuevo repositorio en Git con el nombre task.vigo.lopez.oscar en estado publico.
6.-Copiamos la url de nuestro nuevo repositorio.
7.-Abrimos en Visual Studio Code nuestra terminal con CONTROL+Ñ.
8.-Iniciamos el repositorio en Visual Studio Code con git init.
9.-Ahora debemos añadir todos los archivos con git add.
10.-Ahora estamos añadiendo los archivos a una lista con el comando git commit -m "Ponemos una descripcion".
11.-Con el comando git remote add origin Y nuestra url subimos los archivos a Git.
12.-Con el ultimo comando git push origin main subimos nuestros archivos.
